Principais formatações

Negrito / Destaque

Nesta frase temos um "termo em negrito" usando a tag <b> (b = bold) "NÃO SEMÂNTICA".

Nesta frase temos um "termo em destaque" usando a tag <strong> "SEMÂNTICA".

Itálico / Ênfase

Nesta frase temos um "termo em itálico" usando a tag <i> "NÃO SEMÂNTICA".

Nesta frase temos um "termo em ênfase" usando a tag <em> "SEMÂNTICA".

Texto Marcado

Podemos criar um "texto marcado" com a tag <mark>

Para marcar o texto com "outra cor" é necessário usar CSS.

Texto grande e pequeno

Estamoscriando um "texto grande" e um "texto pequeno" neste parágrafo.

A tag <big> está obsoleta, mas a tag <small> não está.

Texto Deletado

Podemos marcar um texto como "excluído" para indicar que ele deve ser lido, mas não deve ser considerado usando a tag <del>.

Texto Inserido

Podemos marcar um texto como "inserido" para dar uma êndase e dizer que ele foi adicionado depois usando a tag <ins>. "SEMÂNTICA".

Existe também a tag <u> para "sublinhar" (underline)."NÃO SEMÂNTICA"

Texto sobrescrito e subscrito

Texto sobrescrito é um expoente, por exemplo: "x3" com a tag <sup>

Texto subscrito é como por exemplo: "H2O" com a tag <sub>
